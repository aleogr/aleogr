A comprehensive automation suite for managing my local development environment. This project uses **Infrastructure-as-Code** principles to streamline the configuration of both the host machine and virtualized guest environments.

### Core Objectives

* **Declarative Setup**: Utilizing Ansible to ensure consistent and reproducible configurations.
* **Host Management**: Rapid deployment and maintenance of the physical workstation.
* **VM Orchestration**: Automated lifecycle management for virtualized laboratory environments.
* **Efficiency**: Eliminating manual overhead to allow for seamless environment replication and testing.
